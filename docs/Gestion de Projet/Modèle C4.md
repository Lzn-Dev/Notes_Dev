# Le modèle C4

C4 pour quatre *boites* ou quatre *conteneurs* , du modèle C1 ( le moins détaillé ) au modèle C4 ( le plus détaillé ).

Ressource : https://c4model.com/

## Les différents niveaux

						   ------------Surface------------
								- C1 : **Context**
								- C2 : **Container**
					     ------------Technique------------
								- C3 : **Components**
								- C4 : **Code**
